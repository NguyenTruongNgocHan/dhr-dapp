# DHR DApp (Digital Health Records)

A decentralized application for managing health records using:
- Vyper smart contracts
- Geth local blockchain
- Web3.py (Python backend)
- React frontend

## Architecture
React -> FastAPI -> Web3.py -> Geth -> Smart Contract

## Features
- Register patient & doctor
- Grant/revoke access
- Add/view health records
- Activity logging
